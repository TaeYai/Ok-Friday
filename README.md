# Friday Night Funkin': Tr1Ngle Engine

Features:

	New Input System (not like kade new input system);
		
	Misses info;
	
	Accuracy info;
		
	Customizable keybinds;

	Modifiers;

	Note splashes;

	Change Difficulty in pause menu;

	Downscroll;

	Blue Balled Counter in pause menu;

	Memory Counter;
	
	Colors for Health Bar;
